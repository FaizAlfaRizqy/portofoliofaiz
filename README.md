# Faiz Portfolio

A modern portfolio website built with **Laravel**, designed to showcase my projects, skills, and journey as a backend developer and embedded systems enthusiast.

> This project is also part of my learning journey. Every line of code is written manually as I continue improving my frontend and backend development skills.

---

- Laravel 12
- Blade
- CSS
- JavaScript (planned)
- MySQL (planned)
